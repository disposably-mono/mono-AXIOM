# memory/summaries/

Rolling summaries used when conversation context overflows.
Generated by axiom-brain when a session exceeds its context budget.

Frontmatter schema:
```yaml
---
summary_id: <uuid>
covers_session: <session_id>
covers_range: [start_message_id, end_message_id]
created_at: 2026-05-20T14:30:00Z
---
```
