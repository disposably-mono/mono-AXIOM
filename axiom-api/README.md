# axiom-api

**Layer 5 — HTTP gateway and web dashboard.**

FastAPI-based gateway that exposes the full system to a browser. Chat, memory browsing, RAG upload, job monitoring, settings, and exports — all from one local web app.

## Responsibility

- REST endpoints for every system capability
- Server-sent event streaming for model responses
- Static frontend serving (React or vanilla JS, TBD)
- Authentication and basic settings persistence
- Conversation, vault, and session export

Not yet implemented. Phase 5 (Weeks 16–18).
